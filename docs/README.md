# Documentation Index

Use this index as the entry point for contributor workflows and repository conventions.

- `build-and-test.md`: configure/build/test commands, preset matrix, and CI-parity validation.
- `architecture.md`: module boundaries, target conventions, dependency rules, and current module
  inventory (`math`, `quantities`, `linear_algebra`, `spherical_geometry`).
- `contributing.md`: contribution checklist, quality gates, and PR expectations.
- `linear-algebra.md`: vector/matrix APIs, decomposition behavior, and usage notes.
- `performance-benchmarks.md`: local benchmark setup, baseline capture, and regression comparison.
- `quantities.md`: quantity type model, canonical units, conversion behavior, and usage notes.
- `spherical-geometry.md`: spherical geometry API overview, invariants, and usage notes.
- `style-guide.md`: C++, CMake, testing, and documentation standards for this repository.
